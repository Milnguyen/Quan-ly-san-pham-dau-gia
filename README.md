 🏷️ Auction Product Management System

Hệ thống quản lý sản phẩm đấu giá được xây dựng bằng **Spring Boot**, hỗ trợ quản lý sản phẩm với các chức năng thêm, sửa, xóa và tìm kiếm.

---
 📌 Giới thiệu

Đây là dự án Web Back-end sử dụng:

- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap 5

Ứng dụng giúp người dùng quản lý các sản phẩm đấu giá một cách đơn giản và trực quan.

---

 🚀 Chức năng

- ✅ Hiển thị danh sách sản phẩm
- ✅ Thêm sản phẩm mới
- ✅ Chỉnh sửa thông tin sản phẩm
- ✅ Xóa sản phẩm
- ✅ Tìm kiếm theo tên sản phẩm
- ✅ Lọc theo khoảng giá
- ✅ Giao diện Bootstrap Responsive
